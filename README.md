## 🤖 IA na Programação

**Este projeto marca o início de uma nova forma de estudar e criar:**

- A IA não escreve código por você — ela te ajuda a **aprender mais rápido**
- Use este site como **ponto de partida**, não como destino final
- Questione, adapte, expanda — o conhecimento é construído na prática

> *"IA não substitui o estudo — ela acelera o início da jornada."*
