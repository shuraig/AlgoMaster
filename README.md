## 🎯 Sobre o Projeto

<p align="justify">
O <strong>AlgoMaster</strong> nasceu de uma necessidade real: aprender algoritmos de forma sólida, sem depender de vídeos, e com conteúdo inteiramente em português. O site foi pensado para quem prefere <strong>ler, praticar e construir o raciocínio no próprio ritmo</strong> — sem pausar vídeo, sem rebobinar, sem depender de conexão para assistir aulas.
</p>

<p align="justify">
O ponto de partida é o <strong>Portugol</strong> — pseudocódigo em português usado amplamente no ensino brasileiro de lógica de programação, executável no <a href="https://www.clubedohardware.com.br/forums/topic/1639070">Visualg 3.0</a>. Antes de qualquer linguagem real, você aprende a <em>pensar como um computador</em>. Inspirado na metodologia do Prof. <strong>Gustavo Guanabara</strong> do <a href="https://cursoemvideo.com">Curso em Vídeo</a>.
</p>

> 💡 Todo o conteúdo — estrutura, código, textos, fluxogramas, exercícios e curadoria de referências — foi gerado em uma conversa iterativa com **Claude**, assistente de IA da [Anthropic](https://anthropic.com), e refinado a cada etapa.

## 🧠 Para quem é este material

Este site foi feito especialmente para quem:

- Prefere **ler e praticar** a assistir vídeos
- Está começando do **absoluto zero** em lógica de programação
- Quer entender os conceitos **antes de aprender uma linguagem**
- Estuda por conta própria e precisa de um material estruturado
- Compartilha conteúdo com outros iniciantes

> Você não precisa instalar nada para começar — apenas abra o site. Para executar os algoritmos, baixe o **Visualg 3.0** gratuitamente.

## 🤖 IA na Construção do Projeto

<p align="justify">
Este projeto é também uma demonstração prática do que é possível construir com o auxílio de Inteligência Artificial. Todo o desenvolvimento — HTML, CSS, JavaScript, fluxogramas SVG, exercícios, quiz, textos didáticos e curadoria de referências — foi produzido em uma única sessão de conversa com o <strong>Claude</strong>.
</p>

**O que isso significa na prática:**

- A IA não pensa por você — ela amplifica o que você já sabe direcionar
- Cada seção foi refinada através de feedback humano, iteração e escolhas editoriais
- O conteúdo técnico foi revisado e validado antes de cada publicação
- O resultado é 100% funcional, sem frameworks, sem dependências externas

> *"IA não substitui o estudo — ela acelera o início da jornada."*
